# combinatorial-barrier-option-pricer
A short tutorial on O(N) option pricing of up-and-out options using combinatorial methods.
Note that I made this in tutorial format to teach myself, hence some parts I struggled with are explained in much more detail than others, vice versa. This is NOT meant to be comprehensive.

This program was largely adapted from this paper (up to Ch 3): 
https://math.mit.edu/~apost/courses/18.204_2018/18_204_paper_annikakim.pdf

I recommend reading through it; Ch 4 and above extends the model to double knock-in options.
